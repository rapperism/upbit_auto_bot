# 업비트 자동매매 봇 설계

## 1. 기술 스택
- **언어**: Python 3.11
- **라이브러리**: `pyupbit` (업비트 API 래퍼), `pandas` (데이터 분석), `schedule` (작업 예약)
- **API**: Upbit Quotation API (시세), Exchange API (주문/자산)

## 2. 매매 전략: 변동성 돌파 전략 (Volatility Breakout Strategy)
- **개요**: 래리 윌리엄스가 고안한 전략으로, 가격의 변동폭을 기준으로 돌파 시 매수하고 장 마감 시 매도하는 전략.
- **매수 조건**: `현재가 > (당일 시가 + (전일 고가 - 전일 저가) * k)`
  - `k` 값은 보통 0.5를 사용하나, 최적화 가능.
- **매도 조건**: 다음 날 시가에 전량 매도 (또는 정해진 시간에 매도).
- **추가 필터**: 이동평균선(MA5, MA20)을 활용하여 상승장일 때만 진입하도록 설정.

## 3. 주요 기능
- **자산 조회**: 현재 보유 중인 KRW 및 코인 잔고 확인.
- **시세 조회**: 실시간 현재가 및 캔들 데이터 수집.
- **주문 관리**: 시장가 매수/매도 기능.
- **로깅**: 매매 내역 및 오류 상황을 파일로 기록.
- **백테스트**: 과거 데이터를 기반으로 전략의 수익률 시뮬레이션.

## 4. 보안 및 주의사항
- **API Key 관리**: `.env` 파일 또는 환경 변수를 통해 Access Key와 Secret Key를 안전하게 관리.
- **최소 주문 금액**: 업비트의 최소 주문 금액(보통 5,000 KRW) 준수.
- **슬리피지**: 시장가 주문 시 발생하는 가격 차이 고려.
