# 업비트 자동매매 봇 실행 가이드

## 1. 개요

본 문서는 업비트(Upbit) 거래소에서 파이썬(Python)을 이용한 자동매매 봇을 실행하기 위한 상세 가이드입니다. 변동성 돌파 전략을 기반으로 하며, API 연동, 전략 구현, 백테스트 및 실제 거래 실행 방법을 포함합니다.

## 2. 준비물

*   **Python 3.11 이상**: 파이썬 공식 웹사이트에서 다운로드 및 설치합니다.
*   **Upbit 계정**: 업비트 거래소에 가입하고 본인 인증을 완료해야 합니다.
*   **Upbit API Key**: 업비트 개발자 센터에서 Access Key와 Secret Key를 발급받아야 합니다. [1]
    *   `IP 주소 등록`은 봇을 실행할 서버의 공인 IP 주소로 설정해야 합니다.
    *   `주문` 및 `자산 조회` 권한을 반드시 체크해야 합니다.

## 3. 환경 설정

1.  **프로젝트 폴더 생성 및 이동**

    ```bash
    mkdir upbit_bot
    cd upbit_bot
    ```

2.  **필요 라이브러리 설치**

    ```
    UPBIT_ACCESS_KEY=YOUR_ACCESS_KEY
    UPBIT_SECRET_KEY=YOUR_SECRET_KEY
    ```

    **주의**: `.env` 파일은 외부에 노출되지 않도록 주의해야 합니다. Git과 같은 버전 관리 시스템 사용 시 `.gitignore`에 추가하는 것을 권장합니다.

## 4. 봇 코드 (`upbit_bot.py`)

다음 파이썬 코드를 `upbit_bot.py` 파일로 저장합니다.

```python
import os
import time
import pyupbit
import pandas as pd
from datetime import datetime
from dotenv import load_dotenv

# .env 파일에서 API Key 로드
load_dotenv()
access_key = os.getenv("UPBIT_ACCESS_KEY")
secret_key = os.getenv("UPBIT_SECRET_KEY")

# 업비트 객체 생성
upbit = pyupbit.Upbit(access_key, secret_key)

def get_target_price(ticker, k):
    """변동성 돌파 전략으로 매수 목표가 조회"""
    df = pyupbit.get_ohlcv(ticker, interval="day", count=2)
    target_price = df.iloc[0][\'close\'] + (df.iloc[0][\'high\'] - df.iloc[0][\'low\']) * k
    return target_price

def get_start_time(ticker):
    """시작 시간 조회"""
    df = pyupbit.get_ohlcv(ticker, interval="day", count=1)
    start_time = df.index[0]
    return start_time

def get_ma5(ticker):
    """5일 이동 평균선 조회"""
    df = pyupbit.get_ohlcv(ticker, interval="day", count=5)
    ma5 = df[\'close\'].rolling(window=5).mean().iloc[-1]
    return ma5

def get_balance(ticker):
    """잔고 조회"""
    balances = upbit.get_balances()
    for b in balances:
        if b[\'currency\'] == ticker:
            if b[\'balance\'] is not None:
                return float(b[\'balance\'])
            else:
                return 0
    return 0

def get_current_price(ticker):
    """현재가 조회"""
    return pyupbit.get_orderbook(ticker=ticker)["orderbook_units"][0]["ask_price"]

def run_trading_bot(ticker="KRW-BTC", k=0.5):
    """자동매매 실행 루프"""
    print(f"Trading bot started for {ticker} with k={k}")
    
    while True:
        try:
            now = datetime.now()
            start_time = get_start_time(ticker)
            end_time = start_time + pd.Timedelta(days=1)

            # 09:00:00 ~ 다음날 08:59:50 (장 운영 시간)
            if start_time < now < end_time - pd.Timedelta(seconds=10):
                target_price = get_target_price(ticker, k)
                ma5 = get_ma5(ticker)
                current_price = get_current_price(ticker)
                
                # 매수 조건: 현재가 > 목표가 AND 현재가 > 5일 이동평균선
                if target_price < current_price and ma5 < current_price:
                    krw = get_balance("KRW")
                    if krw > 5000: # 최소 주문 금액 5,000원
                        print(f"Buying {ticker} at {current_price}")
                        upbit.buy_market_order(ticker, krw * 0.9995) # 수수료 고려
            
            else:
                # 장 마감 시 전량 매도
                coin_balance = get_balance(ticker.split("-")[1])
                if coin_balance > 0.00008: # 최소 매도 수량 (BTC 기준 예시)
                    print(f"Selling {ticker} at {get_current_price(ticker)}")
                    upbit.sell_market_order(ticker, coin_balance)
            
            time.sleep(1)
        except Exception as e:
            print(f"Error: {e}")
            time.sleep(1)

if __name__ == "__main__":
    # 실제 실행 시에는 API Key가 설정되어 있어야 합니다.
    # run_trading_bot("KRW-BTC") # 주석을 해제하여 봇 실행
    pass
```

## 5. 백테스트 코드 (`backtest.py`)

다음 파이썬 코드를 `backtest.py` 파일로 저장합니다. 이 코드는 과거 데이터를 기반으로 전략의 수익률을 시뮬레이션합니다.

```python
import pyupbit
import numpy as np

def backtest(ticker="KRW-BTC", k=0.5, count=30):
    """과거 데이터를 기반으로 변동성 돌파 전략 수익률 계산"""
    # OHLCV(Open, High, Low, Close, Volume) 데이터 가져오기
    df = pyupbit.get_ohlcv(ticker, interval="day", count=count)
    
    # 변동폭 계산 (고가 - 저가)
    df[\'range\'] = (df[\'high\'] - df[\'low\']) * k
    
    # 매수 목표가 (시가 + 변동폭)
    df[\'target\'] = df[\'open\'] + df[\'range\'].shift(1)
    
    # 수익률 계산 (매수가 대비 종가)
    # 매수 조건: 고가가 목표가보다 높아야 함
    df[\'ror\'] = np.where(df[\'high\'] > df[\'target\'],
                         df[\'close\'] / df[\'target\'],
                         1)
    
    # 누적 수익률 (HPR)
    df[\'hpr\'] = df[\'ror\'].cumprod()
    
    # 낙폭 (Drawdown) 계산
    df[\'dd\'] = (df[\'hpr\'].cummax() - df[\'hpr\']) / df[\'hpr\'].cummax() * 100
    
    print(f"--- Backtest Result for {ticker} (k={k}, days={count}) ---")
    print(f"Cumulative Return (HPR): {df[\'hpr\'].iloc[-1]:.4f}")
    print(f"Max Drawdown (MDD): {df[\'dd\'].max():.2f}%")
    
    return df

if __name__ == "__main__":
    # 비트코인 최근 30일 백테스트 실행
    backtest("KRW-BTC", k=0.5, count=30)
```

## 6. 봇 실행 방법

1.  **백테스트 실행 (선택 사항)**

    봇을 실행하기 전에 `backtest.py`를 통해 전략의 성능을 미리 확인해 볼 수 있습니다.

    ```bash
    python3.11 backtest.py
    ```

    *   `ticker`: 거래할 코인 (예: `KRW-BTC`, `KRW-ETH`)
    *   `k`: 변동성 돌파 전략의 `k` 값 (0.1 ~ 1.0 사이, 기본값 0.5)
    *   `count`: 백테스트에 사용할 일수

2.  **자동매매 봇 실행**

    `upbit_bot.py` 파일의 `if __name__ == "__main__":` 아래에 있는 `run_trading_bot("KRW-BTC")` 주석을 해제하고 실행합니다.

    ```python
    if __name__ == "__main__":
        run_trading_bot("KRW-BTC") # 주석을 해제하여 봇 실행
    ```

    터미널에서 다음 명령어를 실행하여 봇을 시작합니다.

    ```bash
    python3.11 upbit_bot.py
    ```

    **경고**: 실제 돈이 오가는 거래이므로, 충분한 테스트와 이해 없이 실전 투입은 매우 위험합니다. 소액으로 시작하거나 모의 투자 환경에서 충분히 검증하는 것을 강력히 권장합니다.
